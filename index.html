HTML

HTML or HyperText Markup Language is the standard markup language used to create web pages. It’s what set’s the basic structure of your website.

How is it used?

  • Tags, Attributes and Elements: The stuff that makes up HTML.
  • Page Titles: Titles. For Pages. A difficult concept, we know…
  • Paragraphs: Structuring your content with paragraphs.
  • Headings: The six levels of headings.
  • Lists: How to define ordered and unordered lists.
  • Links: How to makes links to other pages, and elsewhere.
  • Images: Adding something a bit more than text…
  • Tables: How to use tabular data.
  • Forms: Text boxes and other user-input things.

CSS

CSS or Cascading Style Sheets is a style sheet language used for describing the look and formatting of a document written in a markup language. In our case HTML!

How is it used?

Well first of all we need HTML to style up. If the content isn’t there CSS will not make it pop up out of nowhere.

Intro conclusion

With this we wanted to show you one important key concept that you must always remember! HTML is used for meaning and CSS is used only for the presentation. As you saw it, HTML is nothing more than a bunch of tags that structure our content. The visual formatting of that content comes later when we tackle CSS.
