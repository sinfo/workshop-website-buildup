HTML

HTML or HyperText Markup Language is the standard markup language used to create web pages. It’s what sets the basic structure of your website.

CSS

CSS or Cascading Style Sheets is a style sheet language used for describing the look and formatting of a document written in a markup language. In our case HTML!

In conclusion

With this we wanted to show you one important key concept that you must always remember! HTML is used for meaning and CSS is used only for the presentation. As you saw it, HTML is nothing more than a bunch of tags that structure our content. The visual formatting of that content comes later when we tackle CSS.
